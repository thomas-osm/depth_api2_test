This package supports the parsing of AIS messages. AIS, the Automatic Identification System, is a system aiming at improving maritime safety by exchanging messages between ships, other vehicles in particular aircraft involved in search-and-rescue (SAR), and (fixed) base stations. The standard documents describing AIS can be found here.

See class {@link nl.esi.metis.aisparser.AISParser AISParser} for more information on how to use this package.

See the ESI AIS Parser License for more information on the legal aspects.

The parser was used in the Poseidon project, and is improved in the Metis project to better handle uncertain information. Both projects were led by the Embedded Systems Institute. In both projects Thales Nederlands was the carrying industrial partner, and multiple Dutch universities participated.
